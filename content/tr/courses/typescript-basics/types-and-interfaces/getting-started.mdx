# TypeScript'e Giriş

TypeScript, statik tipler ekleyerek JavaScript'i genişletir. Bu ders tiplerin neden önemli olduğunu ve proje kurulumunu anlatır.

- TypeScript düz JavaScript'e derlenir.
- Derleme için `tsc` veya paketleyicinizi kullanın.
- Tipler hataları çalışma zamanından önce yakalamanıza yardım eder.
