# İlkel Tipler

TypeScript, JavaScript ile aynı ilkel tipleri destekler; açık tip açıklamalarıyla.

- `string`, `number` ve `boolean` en yaygın olanlarıdır.
- Açıklama için değişken adından sonra `: tip` kullanın.
- TypeScript çoğu zaman tipleri başlangıç değerinden çıkarabilir.
