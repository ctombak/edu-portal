# Tip Koruyucular

Tip koruyucular, koşullu bloklar içinde tipleri daraltır; böylece TypeScript doğru tipi çıkarabilir.

- `typeof` ve `instanceof` yerleşik koruyuculardır.
- Özel koruyucular `is` anahtar sözcüğünü kullanır.
- Koruyucular çalışma zamanı kontrollerini tip sistemiyle uyumlu hale getirir.
